deviceName --> Any (Real Device) / Actual value (Simulators)
app --> Absolute Path of my app/ipa in my local machine

deviceName --> Any (Real Device) / Actual value (Simulators)
bundleId --> Bundle Identifier given by developer
automationName --> XCUITest
